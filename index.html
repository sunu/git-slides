Developing with git

A short description on how to use git to improve your day-to-day development tasks

Git Basics

Definitions

Blob

the contents of a file

Tree

a directory of blobs or other trees

Commit

  • A tree
  • Zero or more parent commits
  • A message

Commits can be viewed with the git log command.

Branch

A pointer to a commit

Branches can be managed with the git branch command and changed with the git checkout command.

Remote

The location of another git repository

Remotes can be managed with the git remote command.

Head

A pointer to the current commit

The HEAD behaves like a branch, and is moved by the git checkout command.

Index

A cache between the working directory and the repository

The index can be modified with the

Repository

  • A repository of blobs and trees
  • A list of branches
  • A list of remotes
  • An index
  • A HEAD

The local repository lives in the .git directory.

Basic Commands

Clone

Copies the contents of a repository 


$ git clone git@github.sky:zeal/ProjectA.git
Cloning into 'ProjectA'...
remote: Counting objects: 53, done.
remote: Compressing objects: 100% (32/32), done.
remote: Total 53 (delta 9), reused 49 (delta 5)
Receiving objects: 100% (53/53), 4.68 KiB, done.
Resolving deltas: 100% (9/9), done.

Add

Adds changes to the index 

$ git add changed-file

Status

Lists information about the workspace, index, and repository 


$ git status
# On branch new
# Changes to be committed:
#   (use "git reset HEAD <file>..." to unstage)
#
#   new file:   updated-file
#

Commit

Creates a commit with the contents of the index 


$ git commit -m 'Added updated-file'
[master 1797f98] Added updated-file
 1 files changed, 1 insertions(+), 0 deletions(-)
 create mode 100644 updated-file

Push

Pushes new commits on a branch to a remote 


$ git push origin master
Counting objects: 4, done.
Delta compression using up to 8 threads.
Compressing objects: 100% (2/2), done.
Writing objects: 100% (3/3), 276 bytes, done.
Total 3 (delta 1), reused 0 (delta 0)
To git@github.sky:zeal/ProjectA.git
   0c73cb9..1797f98  master -> master

Fetch

Synchronizes the local repository with a remote repository 


$ git fetch
remote: Counting objects: 3, done.
remote: Compressing objects: 100% (2/2), done.
remote: Total 2 (delta 1), reused 1 (delta 0)
Unpacking objects: 100% (2/2), done.
From github.sky:zeal/ProjectA
   1797f98..858ec06  master     -> origin/master

Creating a branch

$ git checkout -b my-branch
